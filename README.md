# odin-form
This is a beginner HTML & CSS project made via the Odin Project course.
This exercise was the occasion to practice on forms HTML & CSS layout.

## Elements studied through this project :
- Wireframe
- Advanced CSS selectors
- HTML forms configuration
- CSS properties
- CSS functions
- Element position and overlay
- Forms validation

## Many mistakes made through this project, this serves as a reminder on element to consider for future project :
- Non atomic git commit were made
- CSS files were poorly organized at first
